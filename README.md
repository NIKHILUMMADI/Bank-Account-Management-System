# Bank Account Management System (Java)

## Objective
A simple Java-based banking system demonstrating:
- Encapsulation
- Inheritance
- Polymorphism

## Technologies Used
- Java
- OOP Concepts

## Classes
- BankAccount (Abstract Class)
- SavingsAccount
- CheckingAccount
- BankManager (Main Class)

## Features
- Deposit and Withdraw
- Interest Calculation (Savings Account)
- Overdraft Facility (Checking Account)
- Polymorphic behavior using parent references
